# Recipe Box - Flask Web App

A simple web application to store, view, and manage your favorite recipes. Built with Flask.

## Features
- 📖 **Browse Recipes** - View all recipes on the homepage
- 🔍 **Recipe Details** - See full ingredients and instructions
- ➕ **Add Recipes** - Submit new recipes with a simple form
- 👤 **User Collections** - Each user has their own recipe box
- ⭐ **Favorites** - Mark recipes as favorites (add/remove)
- 🏠 **Personal Dashboard** - View your added recipes and favorites

## Tech Stack
- **Backend**: Python, Flask
- **Frontend**: HTML, Jinja2 templates
- **Version Control**: Git

## Installation
1. Clone the repository: `git clone https://github.com/CorbinBlackard/Flask_Recipe_App.git`
2. Navigate to project: `cd Flask_Recipe_App`
3. Install Flask: `pip3 install flask`
4. Run the app: `python3 app.py`
5. Open browser to `http://127.0.0.1:5002`

## Usage
- Browse all recipes on the homepage
- Click any recipe to see full details
- Add new recipes using the "Add Recipe" link
- Mark favorites with the ⭐ button
- View your personal recipe collection
